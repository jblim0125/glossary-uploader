# 데이터 패브릭 사전 업로더  

## 개요

이 프로그램은 정부에서 제정한 공공데이터 표준 용어(or CSV/EXCEL 형태의 사전 데이터)를 데이터 패브릭 서버의 사전으로 업로드(등록)하는 프로그램이다.  

## 요구사항  

1. 데이터 패브릭 서버  
2. Python 3.10 이상  
3. Docker(준비 중)  
    도커를 활용한 사전 업데이트 기능은 준비 단계이다.
